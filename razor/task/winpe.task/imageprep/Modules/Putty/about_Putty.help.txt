PuTTY PowerShell Module

PowerShell functions for PuTTY (http://putty.org/) and companion set of tools.


SHORT DESCRIPTION

This module provides powershell wrapper functions for the following PuTTY tools.

  * PuTTY     - Terminal Emulator for SSH/Telnet/Raw/Rlogin/Serial sessions.
  * PLink     - Command-Line utility to run SSH commands on a remote host.
  * PScp      - Command-Line utility to copy files to a remote host using SSH.
  * Pageant   - SSH authentication agent to hold SSH keys in protected memory.
  * PuTTYGen  - Key Generator to generate SSH key-pairs for putty, plink, etc.


LONG DESCRIPTION

This module contains a collection of powershell functions that wrap the common
putty command line utils to create ssh/telnet/raw sessions.

The documentation on the usage of these tools should be taken from the 
function documentation. The table lists the functions that wrap the tools.

   --------------- ------------------- ---------------------------------------
  | Tool          | CmdLet            | Description                           |
  | ------------- | ----------------- | --------------------------------------|
  | putty.exe     | Invoke-PuTTY      | Start a new PuTTY Session             |
  |               | putty             | Alias for Invoke-PuttySession         |
  |               |                   |                                       |
  | plink.exe     | Invoke-Plink      | Run a command on a remote host        |
  |               | plink             | Alias for Invoke-Plink                |
  |               |                   |                                       |
  | pscsp.exe     | Invoke-PScp       | Copy files to/from a remote host      |
  |               |                   |                                       |
  | pftp.exe      | *TODO*            |                                       |
  |               |                   |                                       |
  | pageant.exe   | Start-Pageant     | Start the SSH agent process in        |
  |               |                   |  the system tray                      |
  |               | Stop-Pageant      | Stop the SSH agent process            |
  |               | pageant           | Alias for Start-Pageant               |
  |               |                   |                                       |
  | puttygen.exe  | Invoke-PuttyGen   | Start the GUI tool to generate keys.  |
  |               | puttygen          | Alias for Invoke-PuTTYGen             |
  |               |                   |                                       |
  | puttytel.exe  | *TODO*            |                                       |
   --------------- --------------- -------------------------------------------

For authotitative documentation that describes the workings of the tools, the
protocols, configuration files - refer to the following sources
  
  * 'PuTTY User Manual' http://the.earth.li/~sgtatham/putty/0.63/htmldoc/

TROUBLESHOOTING TIPS

  * Use the -Verbose parameter to the calls to have the commandline emitted.
